First words

As well go through examples, Ill make sure that you have a small set of words so you can understand examples properly. But before learning any word, lets make wording system clear.

Yenghi manages words as particle collages. Thus, its not properly said words that we generally use, but glued ideas together in order to make sense. A single idea can hold itself different meanings, thats why using simply ideas can make the language really hard to understand. When using simply the idea, we need to accord it regarding the Kanaka. Knowing that each ideas are built on meaning full roots is even more helpful.

Note: There is a small difference between a root and an idea. Each roots ends by the sound un. If you need to use a root in a sentence, you first have to translate it to an idea. To do so, you will just need to change the last sound to an. For people who knows reading Arabic scripts, its dropping to . Yenghi doesnt need to add the after a Fetha-Tanween, as it uses a different reading-system.

Also, know that Yenghi does something extremely uncommon with some words ; It takes a root from a language, changes its pronounciation, and puts it to the Nahaan Kanaka. This is what Yenghi calls Qardash kki ( ), meaning brother root. This si something extremely common in Yenghi, as the Legend of The Great Naame used to create roots on top of stranger roots.

Note: In the Great Turkic world, Turkish people uses iyi word, as Eastern-Turkic people uses Yahshi, while the original one was edg. Yenghi prefered to keep all of them, and they all have the exact same meaning. Dont forget to add the -un at the end when using these words as roots. Change it to -an when using it as an idea.
