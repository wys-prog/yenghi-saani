# Pronouns

As we saw in the overview chapter, the Yenghi language is **strongly typed** when it comes to grouping words by their origins. This system also affects pronouns: **each Kanakaï has its own pronoun set**, and the Kanakaï category is always visible in the word’s form.

This does **not** mean every word changes drastically, but it means that the Kanakaï identity of a word is always recognizable, even out of context.

---

## **Turki Kanakaï (كاناكايى تۈركى)**

Turki Kanakaï pronouns are the simplest for Turkic speakers.
It contains two versions:

* **Normal pronouns**
* **Honorific pronouns**

---

### **Normal pronouns (ساد زمر — saad zamr)**

| Yenghi          | English       |
| --------------- | ------------- |
| men (من)        | I / me        |
| sen (سن)        | you           |
| ung (ئۇڭ)       | he / she / it |
| biz (بز)        | we            |
| siz (سز)        | you (plural)  |
| unglar (ئۇڭلار) | they          |

> **Note:** *unglar* = *ung* + *-lar* (plural).

---

### **Honorific pronouns (ھۆرمەتلىك زمر — Hörmätlik zamr)**

| Yenghi          | English       |
| --------------- | ------------- |
| meng (مڭ)      | I / me        |
| seng (سڭ)      | you           |
| ong (ئوڭ)       | he / she / it |
| bizai (بزاي)    | we            |
| sizai (سزاي)    | you (plural)  |
| onglar (ئوڭلار) | they          |

> **Note:** Here, we added 'g', but removed the single ن to "promote" it to ڭ.

---

## **Atta Kanakaï (كاناكايى ئاتتا)**

Used mainly in proverbs, poetry, and wise sayings.
No honorific system, but still slightly distinct from Turki.

| Yenghi           | English       |
| ---------------- | ------------- |
| màng (مڭ)       | I / me        |
| sàng (سڭ)       | you           |
| hong (ھوڭ)       | he / she / it |
| bàzai (بزاي)     | we            |
| sàzai (سزاي)     | you (plural)  |
| honglar (ھوڭلار) | they          |

---

## **Arabiyya Kanakaï (كاناكايى ئاربييا)**

| Yenghi     | English       |
| ---------- | ------------- |
| man (مان)  | I / me        |
| san (سان)  | you           |
| hun (ھۇن)  | he / she / it |
| nehn (نھن) | we / us       |
| kum (كۇم)  | you (plural)  |
| hum (ھۇم)  | they          |

> **Note:** These are inspired by Arabic pronouns.
